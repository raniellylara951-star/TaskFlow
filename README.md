# ✅ TaskFlow

> Aplicativo de lista de tarefas moderno e intuitivo desenvolvido para ajudar na organização do dia a dia.

---

# 📖 Sobre o Projeto

O TaskFlow é um aplicativo criado para auxiliar usuários na organização de tarefas diárias de forma prática e eficiente.

---

# ✨ Funcionalidades

- ✅ Adicionar tarefas
- ✅ Editar tarefas
- ✅ Excluir tarefas
- ✅ Marcar como concluída
- ✅ Modo escuro
- ✅ Interface responsiva

---

# 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub

---

# ⚙️ Requisitos do Sistema

## Requisitos Funcionais

- RF01 — O sistema deve permitir adicionar tarefas.
- RF02 — O sistema deve permitir editar tarefas.
- RF03 — O sistema deve permitir excluir tarefas.
- RF04 — O sistema deve permitir marcar tarefas como concluídas.

## Requisitos Não Funcionais

- RNF01 — O sistema deve possuir interface responsiva.
- RNF02 — O sistema deve ser intuitivo.
- RNF03 — O sistema deve funcionar em diferentes navegadores.
