# Modo Escuro

**COMO** usuário,

**QUERO** ativar ou desativar o modo escuro

**PARA** melhorar a visualização da aplicação em diferentes ambientes.

**Critérios de Aceitação**

- Deve existir uma opção para ativar o modo escuro.
- Deve existir uma opção para desativar o modo escuro.
- As cores da interface devem ser alteradas corretamente ao mudar o tema.
- O conteúdo deve permanecer legível em ambos os modos.
