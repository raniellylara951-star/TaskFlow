# Marcar Tarefa como Concluída

**COMO** usuário,

**QUERO** marcar uma tarefa como concluída

**PARA** acompanhar meu progresso.

**Critérios de Aceitação**

- Deve existir uma opção para concluir a tarefa.
- A tarefa concluída deve ser destacada visualmente.
- O status da tarefa deve ser atualizado imediatamente.
