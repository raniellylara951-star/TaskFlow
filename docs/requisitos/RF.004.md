# Seleção de Tarefas

**COMO** usuário,

**QUERO** selecionar uma tarefa da lista

**PARA** visualizar ou gerenciar suas informações de forma mais prática.

**Critérios de Aceitação**

- Deve ser possível selecionar uma tarefa com um clique.
- A tarefa selecionada deve ser destacada visualmente.
- Apenas tarefas existentes podem ser selecionadas.
- O sistema deve permitir alternar a seleção entre tarefas.
