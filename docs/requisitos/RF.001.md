# Cadastro de Tarefas

**COMO** usuário,

**QUERO** cadastrar uma nova tarefa

**PARA** organizar minhas atividades diárias.

**Critérios de Aceitação**

- Deve ser possível informar o nome da tarefa.
- Deve ser possível informar uma descrição da tarefa.
- O sistema não deve permitir tarefas sem nome.
- A tarefa deve ser adicionada à lista após o cadastro.
